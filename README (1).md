# Seven Wonders of the World — Interactive Carousel

A visually rich, animated carousel showcasing all 7 Wonders of the World. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks or dependencies needed.

## ✨ Features

- **Smooth 700ms animations** — images, background colors, blur, scale all crossfade simultaneously
- **Tap/click any wonder** to open an info panel with full description
- **Arrow buttons** + **dot indicators** to navigate
- **Swipe support** on mobile (left/right)
- **Keyboard navigation** — Arrow keys + Escape
- **Ghost text** in the background that matches each wonder
- **Per-wonder background color** that transitions smoothly
- **Fully responsive** — works on mobile and desktop

## 🗂️ File Structure

```
sevenwonders/
├── index.html   ← Main page structure
├── style.css    ← All styles and animations
├── main.js      ← Carousel logic, data, interactions

```

## 🚀 How to Run

Just open `index.html` in your browser — no build step required.

Or host it directly on GitHub Pages:
1. Push this folder to a GitHub repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://yourusername.github.io/repo-name/`

## 🎮 Controls

| Action | Result |
|---|---|
| Click center image | Open info panel |
| Click side image | Rotate that wonder to center |
| Arrow buttons | Navigate prev / next |
| Dot indicators | Jump to any wonder |
| Swipe left/right | Navigate (mobile) |
| ← → Arrow keys | Navigate (keyboard) |
| Escape key | Close info panel |
